About Mobile Robot Framework

The Mobile Robot Framework Project is conducted by students as a project work at the Friedrich-Alexander University Erlangen-NÃ¼rnberg. The Goal of this project is to develop a framework for a Raspberry Pi powered robot. The framework should serve the purpose of being a demonstrable and convincing proof of the possibilities of platform-independent mobile apps. It enables a broad variety of devices to interact physically and virtually with their surrounding environment through the robot.

For further information about the AMOS Project please have a look at the website of the Professorship for Open Source Software, Computer Science Department of the Friedrich-Alexander University Erlangen-Nuremberg and for information about the industry partner Basys GmbH please have a look at the website of the develop group.

On the Github page you can see the source files and some more information about this project.
